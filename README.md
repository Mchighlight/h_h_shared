# h_h_shared

This repository is intended to store media files and other large assets.

## Purpose
- Central location for shared media and heavy files
- Keep project repositories lightweight by separating bulky assets

## Folder structure
- /incoming: drop new files here before sorting
- /media/images: photos, graphics, artwork
- /media/video: raw and edited video
- /media/audio: music, voice, SFX
- /media/docs: PDFs and other reference docs
- /archives: compressed bundles and backups
- /projects: per-project subfolders
- /_meta: inventories, checksums, notes

## Notes
- Consider using Git LFS if you plan to version large binaries.
